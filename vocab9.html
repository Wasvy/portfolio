css: display sets whether an element is treated as a block or inline box and the layout used for its children, such as flow layout, grid or flex.
track tag specifies text tracks for audio or video elements.
css: visibility shows or hides an element without changing the layout of a document.
noscript html tag defines an alternate content to be displayed to users that have disabled scripts in their browser or have a browser that doesn't support script
css: font-variant specifies whether or not a text should be displayed in a small-caps font.
css grid: grid-template-rows defines the line names and track sizing functions of the grid rows
css grid: fr unit a unit which represents a fraction of the available space in the grid container
css grid: grid-template-columns defines the number and size of columns in a grid layout
css: text-decoration sets the appearance of decorative lines on text
css grid: grid-template-areas defines a CSS grid by assigning a grid-area name to each element in the template
css grid: grid-area specifies a grid item's size and location within a grid by contributing a line, a span, or nothing (automatic) to its grid placement, thereby specifying the edges of its grid area.
